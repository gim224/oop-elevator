## [의식의 흐름]

- 해당 엘레베이터가 설치될 건물이 총 몇 층일지는 예측할 수 없으나, 엘베는 처음 만들 때 최저층부터 최고층까지의 버튼(number)을 구성한다.
- 건물에 0층은 없다.
- 버튼을 누르면 엘베는 해당 층으로 이동해야한다. 
(엘베가 이동하는 것은 엘베 자신의 책임일까? 도르레..?)
